# `regenerateShareToken`

- Type: `mutation`
- Returns: `Dashboard!`
- Deprecated: no
- Source endpoint: `https://prod-gql.hive.com/graphql`

| Arg | Type | Default | Description |
|---|---|---|---|
| `_id` | `ID!` | - | Dashboard ID to regenerate share token |
