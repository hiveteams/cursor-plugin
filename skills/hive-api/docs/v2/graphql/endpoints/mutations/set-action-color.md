# `setActionColor`

- Type: `mutation`
- Returns: `Action`
- Deprecated: no
- Source endpoint: `https://prod-gql.hive.com/graphql`

## Description

Set the color of an action to be used in the timeline view.

## Arguments

| Arg | Type | Default | Description |
|---|---|---|---|
| `actionId` | `ID!` | - | - |
| `actionColor` | `String!` | - | - |

## Signature

- `setActionColor(actionId: ID!, actionColor: String!)` -> `Action`
